Maintenance Activator Elementor is use for activate elementor maintenace mode from wordpress dashboard by just one click.

== Description ==

Do you want to turn on and off the maintenace mode in a easy way. Maintenace Activator Elementor help you to active maintenace mode with in one click from dashboard.
You can also see the status of maintenace mode on dashboard. 
This plugin works only for Elementor users.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for Maintenance Activator Elementor'
3. Activate Maintenance Activator Elementor from your Plugins page.

= From WordPress.org =

1. Download Maintenance Activator Elementor.
2. Upload the Maintenance Activator Elementor' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Maintenance Activator Elementor from your Plugins page.

= Once Activated =

1. Visit Maintenance Mode' and copy paste the database table id which is showing on Maintenance Mode Page.
2. Make sure you already active the elementor plugin without elementor this plugin will not works.
3. If you had not active Elementor Maintenance Mode before so for the first time use it from the Elementor->Tools.
